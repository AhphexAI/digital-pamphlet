# Scene #9: 그날의 장면들
**대학연합 사진동아리 '그날' 9기 정기 사진전 모바일 리플렛**

대학연합 사진동아리 ‘그날’의 아홉 번째 이야기, 첫 번째 공식 전시회인 [Scene #9: 그날의 장면들]을 위한 공식 모바일 리플렛 웹사이트입니다.

## Live Demo
- URL: https://AhphexAI.github.io/digital-pamphlet
- Short URL: bit.ly/Scene9-exhibit

---

## 주요 기능
- **Exhibition Chapters**: 4개의 챕터별 기획 의도 및 참여 작가 라인업 확인
- **Interactive Floorplan**: 전시장 도면 및 동선 확인 (모달 팝업)
- **Navigation**: 네이버 지도, 카카오맵 연동을 통한 전시장 길 찾기
- **Community**: 공식 인스타그램 및 네이버 카페 링크 연결
- **Mobile Optimized**: 현장 방문객을 위한 반응형 모바일 인터페이스

## 🛠 Tech Stack
### Frontend
- **HTML5**: Semantic Markup (`details`, `summary` 등을 활용한 구조화)
- **CSS3**: Responsive Design (Flexbox 기반 모바일 최적화), Custom Properties(CSS 변수) 활용
- **JavaScript (Vanilla JS)**: Modal UI 및 팝업 시 배경 스크롤 제어 로직 구현
- **Web Font**: Pretendard (CDN을 통한 고해상도 타이포그래피 적용)

### DevOps & Tools
- **Deployment**: GitHub Pages (Branch: `mobile`)
- **Version Control**: Git, GitHub Desktop
- **Editor**: Visual Studio Code
- **URL Management**: Bitly (Custom Short Link 생성 및 QR 연결)

## 프로젝트 구조
- `index.html`: 메인 구조
- `assets/style.css`: 모바일 최적화 디자인
- `assets/images/`: 포스터, 도면, 로고 리소스
