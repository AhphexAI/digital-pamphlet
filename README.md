# Scene #9 – Digital Exhibition Pamphlet

모바일 환경에서 사진전 정보를 제공하기 위한
QR 기반 웹 전시 안내 페이지입니다.

## 🔹 Project Overview
- 동아리 사진전 오프라인 전시와 연계
- 종이 팜플렛 대체 목적
- 모바일 최적화 웹 UI 직접 설계 및 구현

## 🔹 Tech Stack
- HTML5
- CSS3
- GitHub Pages

## 🔹 Features
- 메인 홈 + 하위 정보 페이지 구조
- 모바일 반응형 레이아웃
- 전시 정보 / 작가 목록 / 서문 제공
